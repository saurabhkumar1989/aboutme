Text Classifier
Primary objective of this project is to build a machine learning based text classifier system to find out the traffic-related incidents in the social media posts. More

System Description

The overall system is composed of four parts.

  • The first section involved data collection using Twitter Rest API.

  • In the second stage, the task of noise removal was performed. In this step, certain parts of the tweets will not provide any relevant information like URL (Universal Resource Locator), emoticons and special characters like hashtags (#), at the rate (@), hyphen, etc. which are to be removed from the tweets.

  • In the third step, the filtered text is converted into a vector by using Bag of Words (BOW).

  • In final stage the newly generated feature vector is fed to a supervised classifier algorithm i.e. Neural Network Algorithm

Outcome

Though with the smaller data set the above classifier was working fine but with the bigger data set, system performance was poor and the reason behind it the feature extraction algorithm as Bag of Word ignore the semantics of sentences. And that motivates me to find some better feature extraction algorithms and that search landed me to deep learning methods.
